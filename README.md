# Analizador Estadístico en Python

Programa que calcula medidas estadísticas básicas a partir de una lista de números.

## Medidas calculadas
- Media
- Mediana
- Moda
- Varianza
- Desviación estándar

*Aplica **matemáticas reales**  
*Usa librerías estándar de Python  
*Es fácilmente ampliable (gráficas, archivos, etc.)
